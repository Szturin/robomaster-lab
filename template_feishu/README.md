# template_feishu

这个目录用于存放可直接复制到飞书 Wiki / 飞书文档中的 Markdown 模板。

建议用途：

- `wiki_home.md`：飞书 Wiki 首页模板
- `member_development.md`：成员培养补充说明模板
- `tech_docs_index.md`：技术资料目录模板
- `new_member_guide.md`：新成员入门模板
- `recruitment_detail.md`：招新详细说明模板
- `project_competition.md`：项目 / 比赛延伸阅读模板

使用方式：

1. 打开对应 `.md` 文件
2. 复制内容到飞书 Wiki 页面或飞书文档
3. 按战队实际情况替换链接、联系人、时间和说明
